# Contributing to Calculator-2026-Group4

## Getting Started

### Requirements:
*   **Java 25**
*   **Maven**

### To work on this project

Clone the repository:
```bash
git clone https://github.com/Wawilski/calculator-2026-group3.git
```

In the project directory, you should be able to build the project and run the tests:

```bash
mvn clean install
```

## Contributing


1.  **Fork** the repository on GitHub.
2.  Implement you feature or your bugfix.
3.  **Commit** your changes and **push** them to your fork.
4.  Submit a **Pull Request** to the `main` repository.

## Testing

We use **JUnit 5** for unit testing and **Cucumber** for BDD.

*   Every new feature **must** be accompanied by tests.
*   Run tests locally before submitting a PR:
    ```bash
    mvn test
    ```
